## 自动播放系统重写

**改动文件：**
- `addons/konado/scripts/dialogue/knd_dialogue_manager.gd` — 重度重写
- `scripts/konado/test_dialogue_screen.gd` — 轻度修改

**核心变化：**
1. 状态变量从 5 个缩减为 2 个：`_auto_play_enabled: bool` + `_auto_play_delay: float`
2. 删除了 `autoplay`、`_paused`、`autoplayspeed` 三个旧变量
3. 删除了 `start_autoplay()`、`pause_autoplay()`、`resume_autoplay()`、`_on_autoplay_timer_timeout()` 四个旧方法
4. 新增 `set_auto_play()` / `set_auto_play_delay()` / `notify_panel_opened()` / `notify_panel_closed()` 统一 API
5. 面板阻塞使用 `_panel_open_count: int` 计数器（支持嵌套面板），替代原来的 `_paused: bool`
6. Timer 取消使用代数递增（`_auto_play_timer_generation`），替代 Godot 不存在的 timer.cancel()
7. 配音等待和无配音等待统一通过 `_start_auto_play_timer(delay)` 处理

**关键设计决策：**
- `set_auto_play()` 不写回 settings，避免信号循环。场景按钮是唯一的 settings 写入源
- 面板关闭后延时推进，不立刻跳
- 按钮 toggle 状态始终与 `_auto_play_enabled` 一致，面板打开时文字提示"自动播放"
- 设置面板 auto_mode toggle 通过 `_on_setting_changed → set_auto_play()` 单向同步

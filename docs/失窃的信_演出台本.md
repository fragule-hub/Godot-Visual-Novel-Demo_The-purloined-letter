# 《失窃的信》演出台本

> **改编自**：`docs/失窃的信_v4剧本.md`（v4 修订版）
> **用途**：面向制作人员的演出设计稿，标注 Konado 插件的音乐/表情/语速/特效等演出细节
> **注意**：这不是 `.ks` 文件，不能直接编译运行。转写 `.ks` 时以此为准。

---

### ⚙ 默认设定（不在每句中重复标注）

- **打字机音效**：所有角色对白默认开启（`enable_typing_effect_audio = true`），旁白（▸ 标记段落）默认关闭。
- **背景转场**：场景之间默认使用 `fade`（ALPHA 淡入淡出，持续 1 秒）。
- **Clara 默认朝向**：center（正向）。行动/攀爬场景优先 left（侧向），其次 right。
- **Eve 密闭空间设定**：未确认安全的密闭空间 → `cry`（怕生）；确认安全后 → `neutral`（冷静）；害羞/脸红 → `shy`。

## 标注规范速查表

### ◆ 演出指令符号

| 标注写法 | 含义 | 对应 Konado 实现 |
|----------|------|------------------|
| `◆ 表情：xxx` | 整句对白期间的角色表情 | `actor change Name xxx` |
| `◆ 语速：N cps` | 打字速度（默认约25cps） | `{speed:N}` |
| `◆ 动作：{bounce:Clara,N,H,0.18}` | 立绘跳动（N=次数, H=高度px, 周期固定0.18s） | `{bounce:Name,N,H,0.18}` |
| `◆ 内联变更：{change:Name,state}` | 句中状态切换（表情/身体/方向） | `{change:Name,state}` |
| `◆ 停顿：N秒` | 打字暂停 | `{wait:N}` |
| `◆ 省略号思考：N秒` | 思考停顿+循环省略号 | `{wait_pause:N,3}` |
| `◆ BGM：曲名` | 切背景音乐 | `play bgm 曲名` |
| `◆ BGM停` | 停止背景音乐 | `stop bgm` |

### 舞台指令

| 写法 | 含义 | 对应 Konado |
|------|------|-------------|
| `[角色 进场] 位置：at N` | 角色登场 | `actor show Name state at N` |
| `[角色 退场]` | 角色离场 | `actor exit Name` |
| `[角色 换装/换表情] → xxx` | 全句级状态切换 | `actor change Name xxx` |
| `[角色 移动] → at N` | 移动到新位置 | `actor move Name N` |

### KS 内联文本标签

在对话文本字符串内使用 `{tag:params}` 格式，打字过程中按字符位置触发：

| 标签 | 语法 | 说明 | 示例 |
|------|------|------|------|
| `speed` | `{speed:N}` | 切换打字速度（cps） | `{speed:45}` 加速到45cps |
| `wait` | `{wait:N}` | 打字暂停 N 秒 | `{wait:0.5}` |
| `wait_pause` | `{wait_pause:N,M}` | 思考停顿 + 循环省略号（N秒后可结束，最多M个点） | `{wait_pause:3,3}` |
| `change` | `{change:角色名,状态}` | 句中渐变切换立绘状态 | `{change:Clara,face=happy}` |
| `bounce` | `{bounce:角色名,次数,高度,周期}` | 立绘跳动 | `{bounce:Clara,1,25,0.18}` |

## 角色表情对照表

### Eve（单图立绘，7状态）

| 标注用名 | Konado 状态 | 使用场景 |
|----------|-------------|----------|
| 平常 | `neutral` | 一般对话、冷静陈述 |
| 微笑 | `smile` | 开心、感到安心 |
| 害羞 | `shy` | **仅用于**脸红/羞耻/被逗弄时的尴尬 |
| 怕生 | `cry` | 未确认安全的密闭空间、对陌生环境的恐惧 |
| 惊讶 | `surprise` | 突然信息、意料之外 |
| 发怒 | `angry` | 愤怒、强硬质问、连珠炮输出 |
| 大笑 | `laugh` | （暂未使用） |
| 哭泣 | `cry` | 同上，`cry` 同时用于怕生和悲伤

### Clara（组件化立绘，3重预设：方向 × 身体 × 表情）

> Clara 现有 **28 种面部表情**（由 eyes + mouth 组件合成），3 种方向（center/left/right），2 种身体预设（casual/coat）。
> 方向默认 `center`，身体默认 `casual`（短袖常服），表情默认 `neutral`。
> 
> **3重预设语法**：`preset:dir_xxx`（方向）+ `preset:body_xxx`（身体）+ `face=xxx`（表情）
> **示例**：`preset:dir_left|preset:body_coat|face=happy` = 左向 + 外套 + 开心

#### 方向预设

| 预设名 | 效果 |
|--------|------|
| `preset:dir_center` | 正向（默认） |
| `preset:dir_left` | 面向左 |
| `preset:dir_right` | 面向右 |

#### 身体预设

| 预设名 | 效果 |
|--------|------|
| `preset:body_casual` | 短袖常服（默认） |
| `preset:body_coat` | 穿外套 |

#### 常用组合

| 标注用名 | Konado 写法 | 说明 | 使用场景 |
|----------|------------|------|----------|
| 日常 | `face=neutral`（或用默认） | 正面 + 常服 + 无表情 | **默认登场**、一般对话 |
| 微笑 | `face=happy` | 正面 + 常服 + 开心 | 友好微笑、正常高兴、开心 |
| 左向微笑 | `preset:dir_left\|face=happy` | 左向 + 常服 + 开心 | 侧身微笑 |
| 左向外套 | `preset:dir_left\|preset:body_coat\|face=happy` | 左向 + 外套 + 开心 | 行动前自信（优先 left） |
| 右向外套 | `preset:dir_right\|preset:body_coat\|face=happy` | 右向 + 外套 + 开心 | 行动（备选 right） |
| 生气 | `face=angry` | 正面 + 常服 + 生气 | 一般生气、轻度吐槽 |
| 暴怒 | `face=furious` | 正面 + 常服 + 暴怒 | 非常愤怒 |
| 正装发怒 | `preset:body_coat\|face=angry` | 正面 + 外套 + 生气 | 强辩、正经愤怒 |
| 正经侦探 | `preset:body_coat\|face=happy` | 正面 + 外套 + 开心 | 装正经、推理发言 |
| 惊讶 | `face=surprised` | 正面 + 常服 + 惊讶 | 突然发现、meta 震撼 |
| 困惑 | `face=confused` | 正面 + 常服 + 困惑 | 疑惑、不理解 |
| 严肃 | `face=serious` | 正面 + 常服 + 严肃 | 认真、思考 |
| 自信 | `face=confident` | 正面 + 常服 + 自信 | 自信满满、行动宣言 |
| 尴尬 | `face=embarrassed` | 正面 + 常服 + 尴尬 | 尴尬、被揭穿 |
| 脸红 | `face=blush` | 正面 + 常服 + 脸红 | 害羞、心动 |
| 悲伤 | `face=sad` | 正面 + 常服 + 悲伤 | 难过、沮丧 |
| 哭泣 | `face=crying` | 正面 + 常服 + 哭泣 | 哭泣 |
| 奸笑 | `face=smirk` | 正面 + 常服 + 奸笑 | 不怀好意、坏笑 |

> 其余表情（mock/fright/terror/sobbing/unease/tired/exhausted/sleepy/disgusted/nauseating/kiss/soulless/psychotic/stoic/scared）已导出但暂未用于剧本，需要用时可随时添加。

### 背景转场效果

KS 语法：`background <name> <effect>`

| 效果名 | 效果描述 | 时长 | 使用建议 |
|--------|----------|------|----------|
| `none` | 无效果（瞬间切换） | 0s | 开场初始化 |

| `erase` | 擦除效果 | 1s | 回忆/闪回 |
| `blinds` | 百叶窗效果 | 1s | 戏剧性场景切换 |
| `wave` | 波浪效果 | 1s | 梦境/晕眩 |
| `vortex` | 极坐标漩涡 | 1s | 时空扭曲感 |
| `windmill` | 风车效果 | 1s | 特殊转场 |
| `cyberglitch` | 电子故障 | 1s | 科技/故障感 |

## Scene 1-1：暑天的推理部
======================================================================
【背景】test_room fade
【BGM】easygoing
【环境感】旧校舍三楼，蝉鸣、空调、摊开的侦探小说。宁静到无聊的午后。

---
▸ 暮月学园旧校舍三楼，推理部活动室。
▸ 蝉鸣透着玻璃窗传入，空调运行声稳定得让人安心。桌面上摊着侦探小说、旧报纸和一叠根本没人整理过的委托记录。
▸ 暑天的部室就是这样。
▸ 阳光、蝉鸣、空调、小说。只要无人打扰，这里几乎可以算作暮月学园少数安逸的地方。

▸ 窗外传来很轻的一声金属摩擦。
▸ 我翻过一页书。
▸ 窗框轻轻震了一下。
▸ 我继续向下浏览。
▸ 窗户被人从外面推开。
▸ 克拉拉从窗台外探进半个身子，金发被夏风吹得乱翘，脸上挂着十分灿烂、十分不应当出现在三楼窗外的笑容。
---
[克拉拉 进场] 位置：at 1 | 状态：日常
  ◆ BGM：battle（克拉拉突然出现）

克拉拉：「下午好呀，副部长！」
  ◆ 动作：{bounce:Clara,1,25,0.18}

我：「……」

克拉拉：「今天天气这么好，你居然窝在室内看小说？」

我：「我先确认一件事。」

克拉拉：「请。」

我：「那是锁着的窗户吧？」

克拉拉：「诶？有锁吗？」
  ◆ 表情：face=confused（歪头无辜）
  ◆ 动作：{bounce:Clara,1,10,0.18}

我：「有。」

克拉拉：「不是一拉就开了吗？」

我：「那一般叫撬开。」

---
▸ 克拉拉从窗台上轻巧落地，拍了拍手上的灰。
---
[克拉拉 换表情] → 日常
  ◆ 动作：{bounce:Clara,1,30,0.18}

克拉拉：「副部长，你对词汇的定义太保守了。」

我：「我对法律的定义也比较保守。」

克拉拉：「这是社团活动室。」

我：「所以撬开自己社团的窗户也是侦探？」

克拉拉：「保持手感。」

我：「不要把犯罪预备说得像晨练啊！」
  ◆ 语速：45 cps（吐槽加速）

---
▸ 克拉拉把窗户重新合上，动作熟练得让人很难相信她真的知道"锁"的存在。
▸ 暮月学园历史悠久。旧校舍外墙保留了许多管道和排水设施，据说是当年改建时留下的。对普通学生而言，那是怀旧建筑的一部分。
▸ 但……对克拉拉而言，那就是通勤路线的一部分了。
▸ 克拉拉把椅子拖到桌边，坐下时还顺手把我书上的书签抽出来看了看。
---

我：「还我。」

克拉拉：「副部长，推理小说看到一半就用书签，是对记忆力的背叛。」

我：「你昨天还把自己的手机放进冰箱。」

克拉拉：「……那是为了测试密室。」

我：「测试结果？」

克拉拉：「手机冷静了。」
  ◆ 表情：face=smirk（憋笑）
  ◆ BGM：waiting

我：「6。」
  ◆ BGM：easygoing
